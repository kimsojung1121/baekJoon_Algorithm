# A+B

### 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

### 입력
첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

### 출력
첫째 줄에 A+B를 출력한다.

### 예제 입력 1
1 2

### 예제 입력 1
3

### 풀이
```python
A, B = input().split();

print(int(A) + int(B));
```

---

- 입력
  - `input()`을 사용한다.
  - 안내 문구는 괄호 안에 넣는다. 
  ```python
  number = input("숫자를 입력하세요.");
  ```
  - `input()`으로 입력되는 것은 문자로 판단된다.
- 출력
  - `print(문자열 + 문자열)` 이면 `문자열문자열`이 출력된다.
  - `int()`를 이용해 숫자로 변환 뒤 연산한다.
